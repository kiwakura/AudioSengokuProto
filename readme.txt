・実行方法
AudioSengokuProto02.exe を実行してください。

・操作方法
矢印キーの右、左でプレイヤーの位置を移動します。
上キーでプレイヤーの位置からユニットを出撃させます。
ユニットが敵陣にたどりつくと勝利します。勝利後はスペースキーを押すと次の勝負に進みます。
ESCキーでゲームを終了します。

説明：
・プレイヤーと敵で向かい合い、毎ターンランダムに生成されたユニットを敵陣に向けて出撃させます。送ったユニットが一つでも敵陣にたどり着くと勝利です。逆に敵にユニットが自陣にたどり着くと負けです。

・プレイヤーが選べる位置は、右、中央、左の3か所です。
プレイヤーの初期位置は中央です。方向キーの左右で移動し、移動先の位置に応じた効果音がなります。
ユニットが出撃する音、戦闘を開始した音、伝令の声は位置に応じて、左右または中央でなります。

・1ターン毎にランダムにユニットが生成されます。生成されたユニットは上キーを押すことでプレイヤーの位置から出撃します。1ターンは5秒です。1ターン内に出撃しなかった場合、次のターンにユニットは持ち越されます。

・生成されるユニットは現在3種類です。
コモン：アシガル。標準的なユニットです。
レア：キヘイ。足が速く、アシガルに対して強いです。
スーパーレア：ホウヘイ。動きは遅いですが、攻撃力が高いです。

・出撃したユニットは、戦闘中でない敵のユニットと衝突すると戦闘を開始します。戦闘開始時に効果音が鳴ります。自陣に近いと大きく、自陣から離れていると小さくなります。戦闘中は移動を停止します。

・戦闘に勝ったユニットは、敵陣に向かって進軍を再開します。

・戦闘で全滅した場合、自陣に向かって伝令が放たれます。伝令は自陣につくと、戦闘に負けたことを報告します。

・戦闘音が大きくなったり、出撃してすぐに伝令が負けを伝えるとピンチです。その位置に次のユニットを出撃させましょう。


改変用の説明：
GameSettings.cs に調整するパラメータを集めています。
・1ターンの時間
・自陣と敵陣間のマス数
・各ユニットの抽選率
・各ユニットの体力と攻撃力
・戦闘にかける時間（＝ターン数）
・ユニット毎の効果音の種類
・左右中央の効果音のパン値
